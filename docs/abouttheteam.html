About the Team

Graydon Kupfer

Graydon Kupfer is a junior at Penn State Behrend. He is majoring in DIGIT and minoring in Game Development. He is currently a Schreyer Honors student. He enjoys playing video games and watching movies. Stardew Valley is his favorite game, so working on this project was a fun opportunity.

Contributions:

  • Found the files within the game that contained dialogue and converted them into a readable format.
  • Established most of the project's research questions and goals.
  • Made major contributions to the project website.
  • Created txt files for each character in the game and sorted each one's dialogue into their respective files.
  • Helped clean up the XML files.
  • Autotagged the named entities in the XML files for each in-game location.
  • Created the Cytoscape network showing which characters appear in which locations.
